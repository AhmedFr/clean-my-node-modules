# TidyDisk SEO + Blog Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship robots/sitemap/canonical/JSON-LD fixes and a markdown blog under `/blog` whose 10 pre-written articles auto-publish weekly via ISR date gating.

**Architecture:** Markdown files in `site/content/blog/` parsed by a small `lib/blog/` module (gray-matter + remark). All pages that surface posts use `revalidate = 3600` and filter by publish date at render time, so future-dated posts appear automatically. Spec: `docs/superpowers/specs/2026-07-04-blog-seo-design.md`.

**Tech Stack:** Next.js 16 App Router (ISR), TypeScript, vitest, gray-matter, remark + remark-html + remark-gfm.

## Global Constraints

- All work inside `site/` (plus STATUS.html at repo root in the final task).
- Package manager: pnpm. No packages with build scripts are being added.
- Canonical base: `https://tidydisk.app` (existing `SITE_URL` logic in layout).
- Copy rule: NO em dashes in any user-facing copy or article text.
- Byline: "TidyDisk team". Prices: free scan, 19 euro lifetime license.
- Component convention: one folder per component with index.ts barrel; lib code split by responsibility.
- Commit style: `feat(site): ...`, `fix: ...`, `docs: ...`.

```tsx
// JSON-LD structured data. Serialized server-side; escape < to prevent
// </script> breakout from content strings.
export function JsonLd({ data }: { data: object }) {
  return (
    <script
      type="application/ld+json"
      dangerouslySetInnerHTML={{
        __html: JSON.stringify(data).replace(/</g, "\\u003c"),
      }}
    />
  );
}
```

```ts
// A post goes live at midnight UTC on its frontmatter date. ISR pages
// revalidate hourly, so a scheduled post appears within the hour.
export function isPublished(dateStr: string, now: Date = new Date()): boolean {
  return new Date(`${dateStr}T00:00:00Z`).getTime() <= now.getTime();
}
```

```tsx
export const revalidate = 3600;
export const dynamicParams = true;

export function generateStaticParams() {
  return getPublishedPosts().map((p) => ({ slug: p.slug }));
}

export async function generateMetadata({ params }: Props): Promise<Metadata> {
  const post = await getPostHtml((await params).slug);
  if (!post) return {};
  return {
    title: `${post.meta.title} | TidyDisk blog`,
    description: post.meta.description,
    alternates: { canonical: `/blog/${post.meta.slug}` },
    openGraph: { type: "article", url: `/blog/${post.meta.slug}`, title: post.meta.title, description: post.meta.description, publishedTime: `${post.meta.date}T00:00:00Z` },
  };
}
```
